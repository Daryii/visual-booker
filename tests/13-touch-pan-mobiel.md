# 13. Frontend – Touch pan (mobiel)

**Bestand:** `public/js/public.js`

---

## Test 13.1 – Pan op mobiel via touch

**Actie:**
- Open de frontend boekingspagina op een mobiel apparaat of via Chrome DevTools → Toggle device toolbar.
- Zoom in tot 200%.
- Veeg met één vinger over de plattegrond.

**Verwachting:**
- De plattegrond schuift mee met de vinger.
- Spots kunnen nog steeds geselecteerd worden via een korte tik.

**Resultaat:** ☐ Geslaagd 

**Opmerking:** De plattegrond schuift correct mee met de vinger en spots zijn selecteerbaar via een korte tik, maar de UI kan nog verbeterd worden.
